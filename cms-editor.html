Website Editor einrichten

Um Änderungen automatisch zu speichern, benötigst du einen GitHub-Zugang.

Wie bekomme ich einen Token?
  1. Gehe zu github.com/settings/tokens/new
  2. Melde dich mit deinem GitHub-Konto an
  3. Gib einen Namen ein, z.B. Website Editor
  4. Wähle bei "Expiration" einen langen Zeitraum (z.B. 1 Jahr)
  5. Aktiviere die Checkbox repo (für Repository-Zugriff)
  6. Klicke unten auf "Generate token"
  7. Kopiere den Token und füge ihn hier ein

Hinweis: Der Token wird nur lokal in deinem Browser gespeichert.

So funktioniert der Editor

Texte bearbeiten: Klicke auf einen Text (wird orange umrandet) und tippe los.

Bilder ersetzen: Klicke auf ein Bild (wird blau umrandet) und wähle ein neues aus.

Videos ändern: Klicke auf ein Video und gib eine neue URL ein.

Autosave: Änderungen werden automatisch alle 30 Sekunden zu GitHub gespeichert.

Versionen: Du kannst jederzeit zu einer früheren Version zurückkehren.

Tipp: Der Hauptbranch (main) bleibt immer als Backup erhalten!
